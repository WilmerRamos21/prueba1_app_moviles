Bienvenido a Mi Aplicación Proyecto desarrollado con Ionic + Angular + Supabase Esta aplicación integra diferentes funcionalidades modernas como autenticación de usuarios, gestión de perfil, galería de fotos utilizando la cámara del dispositivo y consumo de APIs externas para la búsqueda de personajes de anime y Star Wars.

El objetivo del proyecto es aplicar conocimientos de desarrollo móvil híbrido, navegación entre páginas, consumo de servicios web, almacenamiento local y diseño de interfaces interactivas.
